Multi-Container Microservices Demo

A DevOps-focused demonstration of containerized microservices

Python API Service

Flask-based backend service for data processing and analysis.

Available Endpoints:

GET /api/python/
GET /api/python/health
GET /api/python/data
POST /api/python/data
POST /api/python/process

Node.js API Service

Express-based frontend service with RESTful endpoints.

Available Endpoints:

GET /api/node/
GET /api/node/health
GET /api/node/users
POST /api/node/users
POST /api/node/process-with-python

Monitoring

Access the monitoring dashboards:

Grafana: http://localhost:3001 (admin/admin)
Prometheus: http://localhost:9090
